# Meridian & Co Advisors

A fictional premium professional services firm website created as a Webleee portfolio demo. The project uses Astro, Tailwind CSS, static output, and a Cloudflare Pages-compatible build.

This demo does not claim real clients, certifications, financial results, legal outcomes, or reviews.

## Project Status

- GitHub repo: `https://github.com/zach601/webleee-advisors-demo`
- Cloudflare live link: `https://webleee-advisors-demo.pages.dev`
- Build status: passed
- Pages created: Home, Services, About, Insights, Contact

## Local Setup

```bash
npm install
npm run dev
```

Open the local URL shown by Astro.

## Build

```bash
npm run build
```

The production output is generated in:

```text
dist
```

## Cloudflare Pages Settings

- Project name: `webleee-advisors-demo`
- Framework preset: Astro
- Build command: `npm run build`
- Output directory: `dist`
- Node version: use the Cloudflare Pages default supported Node runtime or set a current LTS runtime in Pages settings.

## Deployment

1. Push this repository to GitHub.
2. Create or connect a Cloudflare Pages project named `webleee-advisors-demo`.
3. Set the build command to `npm run build`.
4. Set the output directory to `dist`.
5. Deploy the site.
